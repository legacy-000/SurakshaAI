### Key Rules

- **ROWID** (BigInt, auto-increment primary key), **CREATEDTIME**, and **MODIFIEDTIME** are auto-generated by Catalyst on every table — do NOT create these manually.
- **VarChar** max length is **250 characters** — use **Text** for longer content.
- **Text** does NOT support Unique or Search Index — use VarChar when you need those.
- **Encrypted Text** does NOT support Unique or Search Index — use only for sensitive fields that don't need indexing.
- **Foreign Key** requires selecting the referenced table during column creation.
- **Search Index: On** improves WHERE clause performance — enable on columns frequently used in filters.
- **PII/ePHI: On** marks the column for data protection compliance — enable on personal identifiable information.
