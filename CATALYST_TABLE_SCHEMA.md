### Key Rules

- **ROWID** (BigInt, auto-increment primary key), **CREATEDTIME**, and **MODIFIEDTIME** are auto-generated by Catalyst on every table тАФ do NOT create these manually.
- **VarChar** max length is **250 characters** тАФ use **Text** for longer content.
- **Text** does NOT support Unique or Search Index тАФ use VarChar when you need those.
- **Encrypted Text** does NOT support Unique or Search Index тАФ use only for sensitive fields that don't need indexing.
- **Foreign Key** requires selecting the referenced table during column creation.
- **Search Index: On** improves WHERE clause performance тАФ enable on columns frequently used in filters.
- **PII/ePHI: On** marks the column for data protection compliance тАФ enable on personal identifiable information.
