# Data Dictionary

## tables

### `users`
Tracks police personnel credentials and roles.

### `cases`
Stores FIR reports, occurrence dates, and types.

### `accused`
Stores offender lists, aliases, and custody status.

### `victims`
Stores case victim profiles and statements.

### `officers`
Stores rank, posting stations, and contact data.

### `crime_patterns`
Maintains spatial radii and Modus Operandi parameters.

### `alerts`
Triggers system warnings for patterns and geofencing breaches.

### `predictions`
Saves forecasting probability scores for target areas.

### `chat_contexts`
Logs chat history transcripts.
